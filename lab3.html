When working with streams, there are three main ways to read data from them:

  • f.get(c) will read a character (including whitespace characters) from f and store it in variable c.
  • getline(f, s) will read a line string from f and store it in variable s.
  • f >> v will read either a non-whitespace character, integer, double, or a word string and store in variable v

Statements like any of the three shown above can be used as the condition of a loop or if statement, the condition will be false if the reading operation failed.
